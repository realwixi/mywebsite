To set the time: Press both buttons simultaneously to enter set time mode. Button 1 switches between hours/minutes. Button 2 increases the selected value.

UML Sequence Diagram: 2-Button Watch Interaction

Interaction Steps:

  • 1. Watch displays current time in normal mode
  • 2. User presses both buttons simultaneously to enter set time mode
  • 3. Watch enters set time mode with hours blinking
  • 4. User can press Button 1 to toggle between hours and minutes
  • 5. User can press Button 2 to increment the blinking value
  • 6. User presses both buttons to exit set time mode
  • 7. Watch returns to normal mode with updated time
